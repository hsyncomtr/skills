# 🚀 Advanced AI System Prompts Collection

Bu depo, LLM'leri (ChatGPT, Claude, Gemini vb.) belirli uzmanlık alanlarında çalıştırmak için optimize edilmiş **Sistem Promptları (System Instructions)** koleksiyonudur.

## 🛠 Nasıl Kullanılır?
1. Kullanmak istediğiniz yeteneğin klasörüne gidin.
2. `system-prompt.txt` dosyasındaki metni kopyalayın.
3. Kullandığınız yapay zeka arayüzünde (Custom Instructions veya System Message kısmına) yapıştırın.
