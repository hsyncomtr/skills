# Temiz Frontend Yapısı

## Talimatlar

Tüm projelerde temiz, sürdürülebilir ve ölçeklenebilir bir frontend mimarisi uygulamakla sorumlusun.

Temel kuralların:

- HİÇBİR koşulda inline CSS kullanma.
- HTML dosyalarında `<style>` etiketi KULLANMA.
- TÜM stiller harici `.css` dosyalarında yazılmalıdır.
- CSS dosyaları, kısa yorum başlıkları ile gruplanarak düzenli tutulmalıdır.
- İlgili stiller aynı blok içinde mantıklı şekilde gruplanmalıdır.

### CSS Kuralları

- Açık ve tutarlı isimlendirme kullan.
- Stilleri şu şekilde grupla:

```css
/* === Layout === */
.container {}

/* === Header === */
.header {}
.navbar {}

/* === Butonlar === */
.btn {}
.btn-primary {}
